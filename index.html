Avanti Anonymizer

PowerPoint Add-in for anonymisering av presentationer

Installation

  1. Ladda ner manifest.xml
  2. Oppna PowerPoint
  3. Ga till Insert → Office Add-ins → Upload My Add-in
  4. Valj manifest.xml-filen
  5. Klicka pa Anonymisera i Home-fliken

Funktioner

  • ✓ Skannar all text i presentationen
  • ✓ Identifierar foretagsnamn, personnamn, e-post
  • ✓ Hittar belopp, procent, telefonnummer
  • ✓ Foreslar anonymiserade ersattningar
  • ✓ Ersatter text med ett klick
